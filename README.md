# fastfood_recognition

This code trains an SVM to classify photographs of common fast foods.

### Prerequisities

You will need Python 2.7 and the following Python packages:
- PIL
- glob
- sklearn
- random
- math
- numpy
- mathplotlib

## Built With

* PyCharm
